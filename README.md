# tui-rs

This is a toy project built to learn more about Ratatui and experiment with terminal UI concepts.

## Resources

- Ratatui tutorials: https://ratatui.rs/tutorials

## Install Dependencies

Use Cargo to fetch dependencies:

```/dev/null/command.sh#L1-1
cargo fetch
```

## Run Tests

Run the test suite with:

```/dev/null/command.sh#L1-1
cargo test
```
